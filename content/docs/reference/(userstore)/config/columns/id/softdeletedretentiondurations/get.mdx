This endpoint gets the default SoftDeleted column purpose ColumnRetentionDurations for a tenant column, one for each column purpose. SoftDeleted data is data that has been deleted but is retained for a specified purpose and duration after deletion. For each retention duration, if the retention duration is a user-specified value, id will be non-nil, and use_default will be false.
