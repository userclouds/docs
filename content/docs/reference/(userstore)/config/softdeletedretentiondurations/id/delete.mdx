This endpoint deletes a specific SoftDeleted ColumnRetentionDuration for a tenant. SoftDeleted data is data that has been deleted but is retained for a specified purpose and duration after deletion. Once the tenant default retention duration has been deleted, it will fall back to the system default to not retain deleted data.
